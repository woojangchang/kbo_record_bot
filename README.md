# KBO 경기결과 텔레그램 봇 만들기

## 기획
- 날짜를 전송하면 해당 날짜의 (롯데) 경기 결과를 가져옴
  - 경기 결과는 statiz 데이터를 스크랩
- 명령어에 따라 다른 결과값을 가져옴(스코어보드, 득점 장면 등)
- AWS Lambda와 API Gateway 이용
- 봇은 텔레그램 이용(무료이며 간단하다.)
